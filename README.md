# Task tracker React Application

This is a simple project and it includes the react ui as well as JSON-server which is used for mock backend.

## Usage

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```

### To build for production

```
npm run build
```
